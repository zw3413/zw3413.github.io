  • Li(xj)=1L_i(x_j) = 1Li​(xj​)=1, when i = j
  • Li(xj)L_i(x_j)Li​(xj​) = 0, when i != j

The cardinal functions serve to “select” or “active” the corresponding yiy_i yi​ values. In the construction of the Lagrange interpolating polynomial, each cardinal function Li(x)L_i(x) Li​(x) is multiplied by a yiy_i yi​. Due to the special properties of the cardinal functions, when x=xix = x_i x=xi​, only the yiy_i yi​ corresponding to Li(x)L_i(x) Li​(x) is “selected”, while all other terms become 0. This ensures that the Lagrange interpolating polynomial passes exactly through all the data points
