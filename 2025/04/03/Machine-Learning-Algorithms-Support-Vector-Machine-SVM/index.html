3. The Support Vector Machine Algorithm

  1. svm算法的核心是要找到分割超平面wx+b=0
  2. w表示分割超平面的法向量
  3. 要想找到分割超平面的法向量，可以找到两个类别中的支持向量
  4. 要找支持向量，使用它们的性质：它们距离分割平面的距离可以被表示为 wx_i+b =1或者-1
  5. 另外我们需要支持向量距离分割超平面的距离最远，这样找到的分割超平面才是最优的分割超平面
  6. 基于以上两点，我们得到了优化问题的目标函数和约束条件，5是目标函数，4是约束条件
  7. 目标函数5，通过变形及带入约束条件4，可以将优化问题转换为找最小2-norm法向量
  8. 这样我们就得到了svm算法的优化问题的目标函数和约束条件
  9. 使用拉格朗日函数来求解这个优化问题，通过将约束条件乘以拉格朗日乘子然后求和后，组合进目标函数构成拉格朗日函数。
  10. 然后就是利用原目标函数是严格凸的，约束时线性的，也就是原优化问题时个凸优化问题，其满足强对偶性，将拉格朗日函数转换为对偶问题，通过求拉格朗日函数对于\omega和b的偏导，并令其等于零，得到对偶优化问题及条件。解对偶优化问题得到拉格朗日乘子(只有支持向量的\alpha_i >0)。再恢复原始变量\omega， 选择任意一个支持向量带入约束条件求得b。这样就解得了分割超平面。
